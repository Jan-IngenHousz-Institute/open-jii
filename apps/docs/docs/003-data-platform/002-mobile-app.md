# Mobile app

After registering, download the openJII app from Play Store (Android only)

### Features
- Connect to MultispeQ via Bluetooth
- Take measurements via MultispeQ
- Smooth process for fast measurements
  - Auto-increment and auto-remember questions
  - Preview raw data and processed data
  - Upload to web platform
- View measurement data
- Turn off MultispeQ
- Inspect MultispeQ battery status

### Using a phone without emailbox
Be sure to have aan [openJII account](../introduction/quick-start-guide).
Log in via ORCID, Github or use the following steps to **log in via email**:
- Download the openJII app on the phone (lets cal it the 'sensorphone') that you want to connect to your sensor
- Open the app and enter your emailaddress
- Now go to a device that has your emailbox, like a desktop, laptop, tablet or (personal) mobile phone and open the email and scroll to the QR code
- Use the sensorphones'camera to scan the QR code 
- You will be logged in into the app

### Minimal specs
- Android 7.0 Nougat (API level 24) or newer
- minimum 2 GB of RAM
