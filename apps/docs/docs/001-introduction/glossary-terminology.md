# Glossary & Terminology

### Experiment
Named container for one or more Flow- or protocol-driven Measurement sessions over time.  

### FAIR Principles
Guidelines to make digital research assets Findable, Accessible, Interoperable, and Reusable.

### Measurement
A single time-stamped result set, plus metadata (who, where, device, protocol, macro, etc.). Often described as a “row” in the data model.

### Macro
A reproducible data pipeline (aka ETL pipeline) step that transforms raw Bronze data into cleaned/curated Silver data for analysis (e.g., a Databricks job/notebook).  

### ORCID iD
Persistent researcher identifier used for login/attribution and contributor disambiguation. ORCID
